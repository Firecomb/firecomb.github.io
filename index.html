CTP 431 Final Project : Simple MML Player with FM Synthesizer

2016 Fall CTP431 Final Project
I want to make some kind of score playing web api. Finding some 'score-like' things, I can find MML code.
MML is Music - Macro - Language. This code lines include note and length information as text.
Those MML Codes are using for in-Game music scoring, like 'Mabinogi' or 'Archeage'.
So I thought it is good to make web api based MML player, to listen the code before adapting it to game.
I referred following pre-made javascript MML player.
https://framesynthesis.com/experiments/synthesis.js/examples/showcase/
Using this parser, I must connect it to my FM Synthesizer(what I did as HW3).
This code make MML code to MIDI signal(midi num/velocity). So I connected this converted midi signal to FM Synthesizer.

Here is my work. https://firecomb.github.io/final project.html
